# Photos des fondateurs

Dépose ici les photos (format JPG ou PNG) pour qu’elles s’affichent sur la page d’accueil et la page Équipe.

**Noms de fichiers utilisés par l’app (ordre affiché) :**

- `aymeric-du-reau.jpg` — Aymeric du Réau, CEO
- `jean-christophe-kourajian-2.jpg` — Jean-Christophe KOURAJIAN, CPO
- `aurelien-duboc.jpg` — Aurélien Duboc, CTO

Les noms et rôles sont définis dans `src/messages/fr.json` et `src/messages/en.json` (`home.foundersList`). L’ordre des fichiers est fixé dans `src/components/home/Founders.tsx` (`imageFiles`).

**Conseil :** format carré ou portrait, min. 256×256 px. Si un fichier est absent, les initiales s’affichent à la place.

**Photo LinkedIn :** ouvrir son profil → clic droit sur la photo → « Enregistrer l’image sous » → enregistrer ici avec le nom attendu.
