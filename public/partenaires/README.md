# Logos partenaires

Déposez ici les logos pour qu’ils s’affichent sur la page d’accueil (section Partenaires) et la page Références. La liste des partenaires est définie dans `src/data/partners.ts`.

**Noms de fichiers attendus (slug + extension) :**

| Slug | Extension par défaut | Fichier |
|------|----------------------|---------|
| carrefour | svg | `carrefour.svg` |
| renault | png | `renault.png` |
| groupama | png | `groupama.png` |
| murex | png | `murex.png` |
| cleyrop | png | `cleyrop.png` |
| safran | png | `safran.png` |
| thales | png | `thales.png` |
| ministere-armees | jpg | `ministere-armees.jpg` |
| quicksign | png | `quicksign.png` |
| fdj | png | `fdj.png` |
| decathlon | png | `decathlon.png` |
| nexity | png | `nexity.png` |
| outscale | png | `outscale.png` |
| ere-conseils | jpg | `ere-conseils.jpg` |

L’extension utilisée est celle définie dans `src/data/partners.ts` (`logoExt`) ou `png` par défaut. Si un fichier est absent, le nom du partenaire s’affiche à la place.

**Conseil :** fond transparent pour PNG/SVG, hauteur ~40 px pour un rendu homogène.
